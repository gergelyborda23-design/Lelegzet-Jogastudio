# Lélegzet Jógastúdió

Egy kis, családias jógastúdió landing oldalának kész webdesignja — Újbuda, Bartók Béla út.
Meleg, földes, prémium hangulat: nyugalom, természetes fény, lassú tempó.

## Megnyitás

- **Élő oldal:** nyisd meg az [`index.html`](index.html) fájlt böngészőben (reszponzív).
- **Teljes oldal render (egy képen):** `Lélegzet - Weboldal design PRO - Borda Gergely.png`

## Design rendszer

- **Paletta:** krém háttér, olívazöld akcentus, agyag pont-szín
- **Tipográfia:** Fraunces (serif címsorok) + Manrope (sans törzs)
- **Részletek:** [`design-system/MASTER.md`](design-system/MASTER.md) és a vizuális arculattábla: [`design-system/brand-board.html`](design-system/brand-board.html)

## Szakaszok

Hero · Óratípusok · Rólunk · Oktatók · Órarend · Bérletek · Visszajelzések · CTA · Footer
